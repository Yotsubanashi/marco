## ğŸš€ Getting Started

### Quick Start
1. Open `inspire.html` in your browser
2. Navigate using the top menu
3. Click "Explore My Vision Board" to start
4. Add your own content to the vision board
5. Customize the About Me page with your info

### Files
- `inspire.html` - Main HTML structure
- `inspire.css` - Complete styling with animations
- `inspire.js` - Interactive functionality
- `INSPIRE-README.md` - This documentation

### Customization

**Change Colors:**
Edit CSS variables in `inspire.css`:
```css
:root {
    --blush-pink: #fbc2eb;
    --lavender: #a6c1ee;
    --mint: #b8e0d2;
    --peach: #ffd5cd;
}
```

**Update Profile:**
Replace the profile image in the About section:
```html
<img src="your-image.jpg" alt="Profile" class="profile-image">
```

**Modify Goals:**
Edit the goal cards in the HTML to reflect your personal goals.

**Add Affirmations:**
Add custom affirmations to the array in `inspire.js`:
```javascript
const affirmations = [
    "Your custom affirmation here",
    // ... more affirmations
];
```
